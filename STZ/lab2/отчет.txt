Ответы на вопросы
1. Чем класс отличается от структуры?
По умолчаю, классы имеют спецификацию доступа - private, в то время как структуры - public.

2. Как запретить посторонним доступ к свойствам класса?
Указать для них спецификатор доступа - private или не указывать спецификатор вовсе (private будет использоваться по умолчанию).

3. Что такое конструктор?
Конструктор - это особый тип метода класса, который автоматически вызывается при создании объекта этого же класса.

4. Какие типы конструкторов вы знаете?
Конструктор по умолчанию
Конструктор с параметрами
Конструктор копирования
Неявно генерируемый конструктор по умолчанию

5. Покажите на примере вызов конструктора.
Class_name my_name; // Создание объекта с помощью конструктора
Class_name my_name("Name", "Age"); // Создание объекта с помощью конструктора с параметрами
