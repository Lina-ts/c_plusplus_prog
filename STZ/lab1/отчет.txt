Написать программу, представляющую собой простейшую систему для управления данными. 
Количество строк: 4. Столбцы: Название группы, Факультет, Количество студентов, Количество
успевающих студентов. Вычислить процент успевающих студентов по всем факультетам.

Ответы на вопросы
1. Дайте определение типу данных «структура». 
Структура - тип данных, определяемый пользователем. Используется для создания структурной переменной.

3. Как объявить структурную переменную? 
Пример объявления структурной переменной:
	struct MyStruct myvar;

4. Для чего предназначена динамическая память? 
Динамическая память позволяет программисту:
	 самостоятельно осуществлять контроль над временем жизни создаваемых им объектов
	 выделять память под объекты, размер которых заранее неизвестен
	 выделять память под объекты, размер которых превышает размер стека
	 динамически выделенная память под переменную/объект не имеет области видимости, что позволяет, не копируя, использовать их в разных частях программы 

5. Что такое «утечка памяти»? 
Утечка памяти - неконтролируемое уменьшение свободной оперативной или виртуальной памяти. Причиной является невыполнение освобождения памяти, которая приводит к накоплению ненужных данных.

6. Какие средства для работы с текстовыми файлами имеются в C++? 
Для работы с файлами в C++ существует заголовок fstream. 
С его помощью возможно:
	создать входной поток для чтения из файла
	создать выходной поток для записи в файл
	открыть входной поток
	закрыть поток
	использовать операции чтения/записи
