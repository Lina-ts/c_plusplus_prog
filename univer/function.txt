1. Функция - совокупность объявлений и операторов, обычно предназначенных для решения определенной задачи.

Сколько бы ф-ций не было, выполенние всегда начинается с main().
Ф-ция может возвращать максимум одно(!) значение.

Определение ф-ции - описание действий, выполняемых ф-цией.
Объявление ф-ции - задание формы обращения к ф-ции. 
Вызов ф-ции

Определение ф-ции задает тип возвращаемого значения, имя ф-ции, объявление переменных и тд.

Общая структура:
[спецификатор_класа_памяти][спецификатор_типа] имя_функции 
	([список_формальных_параметров])
	{ тело_функции }

(спецификатор необязателен, может быть static или extern)
Пример:

int sum (int a, int b) //ф-ция имее 2 параметра - a и b
{
	int s=0;
	while (a <= b)
		s +=a++;
	return s;	//возвращает целое значение 
}

Прототип ф-ции (объявление) не обязателен, но он позволяет компилятору проверять соответствие типов. 
Можно его указывать так:
	int sum (int a, int b);
или так:
	int sum (int, int);
